1. Jelaskan Perbedaan antara pengembangan aplikasi mobile secara hybrid dan pengembangan aplikasi mobile secara native
        Jawaban : 
* Biaya 
Aplikasi mobile native membutuhkan masing-masing developer android dan ios dalam proses development aplikasi. Sedangkan untuk aplikasi mobile hybrid hanya dibutuhkan satu developer untuk dua platform 
* Waktu 
Waktu yang dibutuhkan untuk mengembangkan aplikasi dengan mobile native cukup lama.
* Device features
Aplikasi native mendapat akses penuh ke perangkat mobile karena langsung dispesifikasikan dengan bahasa pemrograman sesuai platformnya. Hal yang sama juga berlaku untuk aplikasi hybrid namun performanya tidak sebaik aplikasi native
* Kecepatan
Aplikasi mobile native lebih cepat beroperasi dibandingkan dengan mobile hybrid yang masih menggunakan pemrograman HTML
* Maintenance
Proses maintenance juga harus dipertimbangkan dalam membangun sebuah aplikasi mobile. Perubahan dalam aplikasi mobile native membutuhkan waktu dan cenderung sulit dalam pengembangannya karena aplikasi harus dikemas kedalam versi baru untuk melakukan update. Setelah itu dipublikasi ke store masing-masing device untuk selanjutnya di unduh dan diinstall kembali.
* Skala
        Aplikasi mobile native sangat disarankan untuk kebutuhan dengan skala besar karena kaya dengan interaksi UI/UX dan performa yang baik. Sedangkan untuk perusahaan dengan skala kecil disarankan menggunakan hybrid dengan alasan waktu dan modal yang terbatas.
